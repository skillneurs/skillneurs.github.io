Un seau (nom anglais : bucket) est un outil utilisé pour transporter de l'eau, de la lave, de la neige poudreuse ou du lait.

Types de seaux[modifier | modifier le wikicode]

  • Seau d'eau : contient un bloc source d'eau.
  • Seau de lave : contient un bloc source de lave.
  • Seau de lait : permet de confectionner des gâteaux ou d'être bu pour annuler les effets de potion, ne peut pas être placé.
  • Seau de neige poudreuse : contient un bloc de neige poudreuse.

Utilisation[modifier | modifier le wikicode]

Transport[modifier | modifier le wikicode]

Le seau est principalement utilisé dans le transport. Une source d'eau ou de lave peuvent être obtenue en utilisant un seau vide dessus, et la source peut être replacée plus tard.

Traite[modifier | modifier le wikicode]

Un seau de lait peut être obtenu en trayant une vache ou une champimeuh.

Chaudron[modifier | modifier le wikicode]

Une source d'eau peut être extraite d'un chaudron plein à l'aide d'un seau vide.

Distributeur[modifier | modifier le wikicode]

Des seaux pleins peuvent être déposés dans un distributeur qui, une fois activé, fera écouler le fluide contenu dans le seau.

Divers[modifier | modifier le wikicode]

Les seaux sont utilisés pour créer des sources infinies d'eau, en en posant trois aligné tels que les sources forme un trait. Un seau d'eau peut être utilisé pour éteindre un incendie ou le joueur lui-même s'il est en feu, pour cela l'eau doit être déposée aux pieds du joueur pour éteindre les flammes.

Le joueur peut créer le cadre du portail de Nether en utilisant des sources de lave et les transformer en obsidienne à l'aide d'eau, une telle opération n'est possible qu'en utilisant des seaux.
